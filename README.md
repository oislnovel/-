# Mimi Reader

Mimi Reader は、Kindle本やカクヨム作品など「自分で正当に利用できる本文」を、PCのイヤホンで聞くための小さなブラウザアプリです。

## できること

- 本文を貼り付けて、日本語向けのブラウザ音声合成で読み上げる
- `.txt` / `.html` ファイルを読み込む
- 音声、速度、高さ、音量を調整する
- 読み上げ位置をチャンク単位で表示する
- 入力内容と設定をブラウザの `localStorage` に保存する

## 大切な注意

このアプリは DRM を解除したり、Kindle やカクヨムから本文を自動取得したりしません。著作権、各サービスの利用規約、作品ごとの公開条件を守って、あなたが利用できる範囲の本文だけを貼り付けてください。

本文はブラウザ内で処理され、サーバーには送信されません。

## PCのイヤホンで聞く手順

1. Windows / macOS でイヤホンを既定の音声出力先に設定します。
2. `index.html` をブラウザで開きます。ローカルサーバーを使う場合は `npm start` を実行し、表示された URL を開きます。
3. ページを開くと最初に表示される「貼り付け入力欄」の大きな枠をクリックします。そこが貼り付け先です。
4. Kindleアプリやカクヨムなどでコピーした本文を、入力欄に `Ctrl + V`（macOS は `Command + V`）で貼り付けます。
5. 「読み上げ」を押します。

## 開発

```bash
npm test
npm start
```

テストは、読み上げ用のチャンク分割ロジックを Node.js 上で検証します。
